# TPO Group Website

Source for the TPO.group website draft, deployed via GitHub Pages.

- **Live URL:** https://robattpo.github.io/tpo-demo-site/
- **Deploy:** any push to `main` is live in ~1 minute
- **Structure:** `index.html` (page markup + app logic) and `assets/` (js, fonts, images)
- **Contact form:** posts to Web3Forms -> info@tpo.group (access key in design-system.js is public by design); hidden `botcheck` honeypot field filters bots
